Week 3 🎜 Encoding Basics for Notation

🎜 Render your MEI file using Verovio in GitHub
🎜 compare 3 XML elements in the MusicXML file to the corresponding ones in MEI.

The element used to provide the name of the score is different in XML than it is to MEI. In XML it is found in the "work" element, where the "work-title" child element contains the name of the piece. Whereas in MEI it is found within the "titleStmt" element in the child element "title". Furthermore, in MEI the "titleStmt" element contains the "composer" element that contains information on who created the piece of music. Whereas in XML it is provided in the "identification" element where "creator type=”composer”"contains the information on the creator. XMl provided the date if creation through the "encoding-date" element within "encoding", compared to MEI which uses the "date isodate" attribute within the "pubstmt" section.
